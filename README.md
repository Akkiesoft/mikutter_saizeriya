# mikutterサイゼリヤプラグイン

## なんこれ
PZ01とかAA15といった暗号でバトルを仕掛けてくる人たちとコミュニケーションを取るためのプラグイン

## インストール方法

```sh
$ mkdir -p ~/.mikutter/plugin
$ cd ~/.mikutter/plugin
$ git clone https://github.com/Na0ki/mikutter_saizeriya
```
